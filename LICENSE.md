# 📄 สัญญาอนุญาตการใช้งาน (License Agreement)

**โครงการ:** ระบบรับเล่มรถ ตรอ. (Rab-Lem-Rot-Tro)  
**เวอร์ชัน:** 3.0.0  
**วันที่:** 2026-04-13

---

## ⚖️ ข้อตกลงการใช้งานซอฟต์แวร์

### 1. การเป็นเจ้าของลิขสิทธิ์
ซอฟต์แวร์นี้เป็นทรัพย์สินทางปัญญาของผู้พัฒนา สงวนลิขสิทธิ์ตามกฎหมายลิขสิทธิ์ พ.ศ. 2537

### 2. การอนุญาตใช้งาน
ผู้พัฒนาอนุญาตให้ใช้งานภายใต้เงื่อนไขต่อไปนี้:

#### ✅ สิ่งที่ทำได้:
- ใช้งานซอฟต์แวร์นี้เพื่อจัดการข้อมูลทะเบียนรถในสถานตรวจสภาพรถ ตรอ.
- ทำสำเนาซอฟต์แวร์เพื่อสำรองข้อมูล (Backup)
- เผยแพร่ตัวติดตั้ง (Installer) ให้แก่ผู้ได้รับอนุญาตใช้งาน

#### ❌ สิ่งที่ไม่ได้รับอนุญาต:
- แก้ไข ดัดแปลง หรือ Reverse Engineering ซอฟต์แวร์โดยไม่ได้รับอนุญาต
- จำหน่าย จ่ายแจก หรือให้เช่าแก่บุคคลที่สามโดยไม่ได้รับอนุญาต
- ใช้ซอฟต์แวร์นี้ในเชิงพาณิชย์โดยไม่ทำสัญญาอนุญาต
- ถอดรหัส License Key หรือระบบตรวจสอบสิทธิ์

### 3. ระบบ License Key
ซอฟต์แวร์นี้ใช้ระบบ License Key เพื่อตรวจสอบสิทธิ์การใช้งาน:
- License Key จะผูกกับเครื่องคอมพิวเตอร์แต่ละเครื่อง (Per-Machine Activation)
- License Key มีอายุตามระยะเวลาที่กำหนดในสัญญา
- การใช้งานโดยไม่ผ่านระบบตรวจสอบสิทธิ์ถือเป็นการละเมิดลิขสิทธิ์

### 4. การรับประกันและความรับผิด
- ซอฟต์แวร์ถูกจัดส่งในลักษณะ "ตามที่เป็น" (As-Is) โดยไม่มีการรับประกันใด ๆ
- ผู้พัฒนาจะพยายามแก้ไขข้อบกพร่อง แต่ไม่รับผิดชอบต่อความเสียหายที่อาจเกิดขึ้น
- ผู้ใช้งานรับผิดชอบต่อการสำรองข้อมูลของตนเอง

### 5. การยกเลิกสิทธิ์
สิทธิ์การใช้งานจะถูกระงับโดยอัตโนมัติหาก:
- ไม่ชำระค่าLicense ตามกำหนด
- ละเมิดเงื่อนไขในสัญญาอนุญาตนี้
- พยายามแก้ไขหรือ Crack ซอฟต์แวร์

### 6. การบังคับใช้
ข้อโต้แย้งใดๆ เกี่ยวกับสัญญานี้ให้อยู่ภายใต้กฎหมายประเทศไทย
